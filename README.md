# 甲骨文消消乐

一个基于甲骨文字符的消除类游戏，使用HTML、CSS和JavaScript实现。

## 游戏说明

- 点击相邻的甲骨文字符进行交换
- 匹配3个或更多相同的字符可以消除
- 每消除一个字符得10分
- 每1000分升一级，共10个关卡
- 每个关卡使用不同的甲骨文字符集

## 如何玩

1. 访问游戏页面：[https://你的用户名.github.io/仓库名](https://你的用户名.github.io/仓库名)
2. 点击相邻的甲骨文字符进行交换
3. 尝试匹配3个或更多相同的字符
4. 达到1000分升级到下一关
5. 挑战10个关卡，通关游戏！

祝您游戏愉快！
